To build a world-class educational and research platform that combines:

- Professional Portfolio
- Physics Academy
- Geoscience Research
- Artificial Intelligence
- Interactive Scientific Computing
- Open-source Software
- Publications
- YouTube Education

The website is not intended to be merely a portfolio.

It is intended to become an ecosystem.

# Guiding Principle

Every feature must satisfy at least one of these goals:

• Better learning

• Better research communication

• Better scientific computing

• Better discoverability

• Better maintainability

If it satisfies none of these,

don't build it.
